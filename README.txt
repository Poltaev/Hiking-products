Приложение предназначено для помощи организаторам походов в группе.

Целью данной работы было сделать приложение, которое снимает с организатора походов, необходимость проводить расчеты еды.
При установке будет условно стартовые данные о еде подготовленные разработчиком, далее пользователь самостоятельно вносит изменения,
которые могут потребоваться в работе. Так же приложение хранит в памяти, то, кто и что несет каждый участник группы, и позволяет убрать
из рюкзака в случае использования той вещи которая раньше лежала в рюкзаке. Таким образом это снимает большое количество расчетов с руководителя походной группы и позволяет ему заниматься более важными вещами.

Использование только базы данных Room, само приложение не нуждается для работы в подключении к интернету. Это очень важно для приложения 
которое должно работать в условиях отсутствия инфраструктуры.

Интерфейс - несколько экранов. Экраны показывают, доступные продукты, снаряжение, раскладка меню полная, раскладка меню и снаряжения по участникам группы, архив предыдущих походов. 
Реализация базы данных с помощью библиотеки Room.
Для реализации прокручивающихся списков использовал RecyclerView.
